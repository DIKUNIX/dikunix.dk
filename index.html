Formål

Foreningen DIKUNIX Foreningens hjemsted er Datalogisk institut ved Københavns Universitet (DIKU), Københavns Kommune. Foreningens formål er at opsætte og vedligeholde EDB-baserede tjenester til støtte af sociale foreninger og studerende i almindelighed på DIKU.
Flere informationer kan findes i vores vedtægter

Tjenester

DIKUNIX udbyder følgende tjenester til den almene studerende på DIKU:

github

DIKUNIX arbejder løbende på en række projekter.
Nærmere detaljer kan findes på vort github depot på adressen https://github.com/DIKUNIX/.
Hvis du selv har en ide et projekt så kontakt os eller duk op til et af vore hackathons.
